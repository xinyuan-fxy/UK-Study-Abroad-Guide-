# 🇬🇧 英国留学指南

> 25年 Fall 入学，一站式英国留学生活手册。开源共创，让每一届新生少走弯路。

## 📚 内容

- **行前准备** — 行李清单、签证攻略
- **落地安顿** — 租房、银行开户、电话卡
- **日常生活** — 医疗 NHS、交通出行、饮食购物、兼职打工、社交娱乐、安全须知
- **学习与毕业** — 学术攻略、毕业去向

## 🛠️ 技术栈

Vite + React + Tailwind CSS + react-markdown

## 🚀 本地运行

```bash
npm install
npm run dev
```
