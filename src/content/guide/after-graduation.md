# 🎓 毕业去向

> 毕业后留英还是回国？早规划早受益。

---

## 🛣️ 毕业后三条路

| 方向 | 说明 |
|------|------|
| **留英工作** | 申请 Graduate Visa 或 Skilled Worker Visa |
| **回国发展** | 参加秋招、利用海归身份优势 |
| **继续深造** | 读博 / 二硕 |

---

## 🇬🇧 Graduate Route（PSW 签证）

本科和硕士毕业可申请 **2 年**，博士毕业可申请 **3 年**。

| 项目 | 说明 |
|------|------|
| 申请条件 | 持学生签证，在英国完成学位 |
| 申请时间 | 学校通知 UKVI 你已完成课程后 |
| 费用 | £822 申请费 + £1,035/年 IHS |
| 工作限制 | 无限制，可做任何工作 |
| 能否转工签 | 可以，找到雇主后可转 Skilled Worker Visa |

> ⚠️ Graduate Visa **不能续签**，到期后必须转其他签证或离境。

## 💼 留英找工作

### 时间线
| 时间 | 动作 |
|------|------|
| 入学 9-10 月 | 开始了解英国求职市场 |
| 11-12 月 | 大公司 Graduate Scheme 陆续开放 |
| 次年 1-3 月 | 申请高峰期，面试集中 |
| 次年 4-6 月 | 部分公司补录，中小企业持续招 |
| 毕业前 | 尽早确定去向 |

### 求职资源
- **学校 Careers Service**：免费改 CV、模拟面试
- **LinkedIn**：英国职场必备，尽早建立 profile
- **Indeed / Glassdoor**：搜索职位 + 看面试经验
- **TargetJobs / Prospects**：专门面向毕业生的平台

---

## 🇨🇳 回国发展

### 优势
- 海归在部分行业有竞争力（金融、咨询、互联网出海）
- 英语能力 + 国际视野是加分项
- 部分城市有海归落户、购房、创业补贴

### 时间线
| 时间 | 动作 |
|------|------|
| 入学当年 9-12 月 | 国内秋招（主要招聘季） |
| 次年 2-4 月 | 春招补录 |
| 毕业回国后 | 参加下一届秋招 |

> 💡 很多公司接受远程面试，不用等到毕业回国才找。

---

## 📦 离开英国前待办清单

- [ ] 提前退租，拿回押金
- [ ] 关停水电网账户
- [ ] 注销银行账户或保留（看需求）
- [ ] 邮寄行李回国（Seven Seas、大西洋快递等）
- [ ] 向学校和 NHS 更新地址
- [ ] 开具《留学回国人员证明》（部分单位需要）
