# 🎉 社交与娱乐

> 留学不只是学习——融入当地生活同样重要。

---

## 🤝 认识新朋友的途径

| 方式 | 说明 |
|------|------|
| **Freshers' Week** | 新生周，社团招新、派对、免费活动，一定要参加 |
| **学生社团（Society）** | 体育、艺术、文化、学术……应有尽有 |
| **Flat/Hall 室友** | 宿舍是最好的社交起点 |
| **语言交换** | 和国际学生互相教语言，交朋友 |
| **志愿者** | 做义工既能锻炼英语又能积累经历 |

---

## 🏛️ 英国特色社交文化

| 场景 | 礼仪 |
|------|------|
| **Pub 文化** | 英国人社交的核心场所，不喝酒也能去喝软饮 |
| **排队** | 英国人对插队零容忍 |
| **Small Talk** | 聊天气是最安全的开场白 |
| **礼貌用语** | Sorry、Please、Cheers 挂在嘴边 |
| **准时** | 迟到很失礼，尤其是正式场合 |

| 节日 | 日期 | 做什么 |
|------|------|--------|
| **Bonfire Night** | 11月5日 | 看烟花、篝火晚会 |
| **Remembrance Day** | 11月11日 | 佩戴罂粟花纪念 |
| **Black Friday** | 11月底 | 疯狂打折购物 |
| **Christmas Market** | 12月 | 圣诞集市，热红酒 |
| **Boxing Day** | 12月26日 | 全年最大打折日 |
| **New Year** | 1月1日 | 伦敦/爱丁堡盛大烟花 |
| **Pancake Day** | 2-3月 | 煎饼日，大学食堂也有 |
| **St Patrick's Day** | 3月17日 | 全城绿色狂欢 |

## 🏠 留学生常见的孤独感

> 这是完全正常的。以下方法可能有帮助：

- 保持和家人的定期视频
- 加入学校的国际学生社群
- 坚持运动（学校健身房很便宜）
- 必要时寻求学校 Counselling 服务（免费且保密）

---

> 🫂 **记住**：每个人都在适应新环境，你并不孤单。
