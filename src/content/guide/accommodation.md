# 🏠 租房与住宿攻略

> 学校宿舍还是校外租房？看完这篇你就知道了。

---

## 🏢 学校宿舍 vs 校外租房

| | 学校宿舍 | 校外租房 |
|------|---------|---------|
| 价格 | £120-250/周 | £80-200/周 |
| 合同 | 灵活，按学期 | 通常一年起租 |
| 便利性 | 近校园，包 bills | 需自己打理 |
| 社交 | 容易交朋友 | 相对独立 |

---

## 🔍 找房渠道

- **Rightmove** / **Zoopla**：最大房源平台
- **SpareRoom**：找合租室友首选
- **UniAcco** / **Student.com**：学生公寓预订
- **学校官网**：学校合作公寓推荐
- **微信/小红书**：华人房东或转租信息

---

## ⚠️ 避坑指南

1. **实地看房**再签合同，或找信任的人代看
2. **确认账单**：房租是否包含水电网气（bills included）
3. **押金保护**：房东必须将押金存入 TDS 等官方保护机构
4. **仔细读合同**：注意 break clause（提前解约条款）
5. **拍照留证**：入住时拍照记录房屋状况，退房避免纠纷
6. **Council Tax**：全日制学生可申请豁免

## 📮 英国地址格式

英国的地址是从小到大写的，和国内相反：

```
Room X, Flat X, Block X           ← 房间号、公寓号、楼号
Building Name, University         ← 建筑名、学校/公司
Street Name                       ← 街道名
Area / District                   ← 区域
City                             ← 城市
County                           ← 郡（部分地址有）
Postcode                         ← 邮编（非常重要！）
United Kingdom                   ← 国家
```

> 💡 **邮编是关键**：英国邮编精确到街道甚至具体建筑（如 `EH14 4AS`），快递、外卖、打车全靠邮编定位，务必记牢！
