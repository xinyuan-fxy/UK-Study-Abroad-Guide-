# 📚 学习与学术攻略

> 英国大学的学习方式和国内大不同，提前了解少踩坑。

---

## 🎓 英国大学教学方式

| 形式 | 说明 |
|------|------|
| **Lecture（大课）** | 教授讲，几百人听，和国内类似 |
| **Seminar（讨论课）** | 15-30 人，讨论阅读材料，**必须发言** |
| **Tutorial（辅导课）** | 1对1或小组，和导师深入交流 |
| **Lab / Workshop** | 理工科实验，艺术类实践 |

> 🔑 Seminar 上的参与度会计入平时成绩，不能只当听众。

---

## 📝 作业与考核

| 类型 | 特点 |
|------|------|
| **Essay（论文）** | 有严格的引用格式（Harvard/APA/MLA） |
| **Report（报告）** | 偏实践，结构更重要 |
| **Presentation** | 小组或个人展示，PPT + 演讲 |
| **Exam（考试）** | 期末闭卷/开卷，部分专业无考试 |
| **Dissertation** | 毕业论文，硕士 1-2 万字 |

---

## ⚠️ 学术诚信（Plagiarism）

英国对抄袭**零容忍**，后果从挂科到开除。

| 红线 | 说明 |
|------|------|
| 抄袭他人 | 直接复制别人的文章 |
| 抄袭自己 | 把之前交过的作业再交一次 |
| 引用不当 | 用了别人的观点但不标注出处 |
| AI 代写 | 各校政策不同，部分学校严禁 |
| 代写代考 | 抓到后果极其严重 |

### 正确做法
- 所有引用都用学校要求的格式标注
- 使用 **Turnitin**（查重系统）提前自查
- 善用学校的 Academic Writing Centre

## 🛠️ 学习工具推荐

| 工具 | 用途 |
|------|------|
| **Grammarly** | 英文写作语法检查 |
| **Zotero / Mendeley** | 参考文献管理 |
| **Notion / OneNote** | 笔记整理 |
| **DeepL** | 翻译（比 Google 准） |
| **Otter.ai** | 录音转文字（上课录音辅助） |

---

## 💡 高分秘诀

1. **读 Reading List**：老师指定的阅读一定得看
2. **善用 Office Hour**：主动找老师问问题，留下好印象
3. **组团学习**：和国际学生组队，既练英语又互助
4. **别攒到最后**：Essay 不可能一晚上写好

---

> 🎯 **核心区别**：英国大学看重批判性思维（Critical Thinking），不是背答案，而是会分析。
