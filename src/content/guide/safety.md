# 🔐 安全须知

> 英国整体安全，但防人之心不可无。

---

## 📞 紧急电话

| 号码 | 用途 |
|------|------|
| **999** | 紧急报警 / 救护车 / 消防（危及生命时） |
| **101** | 非紧急报警（物品丢失、噪音投诉等） |
| **111** | 非紧急医疗咨询（24小时） |
| **112** | 欧盟通用紧急号码（和 999 一样） |

> 📱 到英国后立刻把 999 和 111 存进手机通讯录！

## 🚨 常见安全提醒

### 城市安全
| 提醒 | 说明 |
|------|------|
| 手机防盗 | 街边别拿手机刷，飞车抢夺在伦敦不少见 |
| 夜间出行 | 不走小巷、不戴耳机、结伴而行 |
| 公交地铁 | 包放前面，手机钱包别放后口袋 |
| 酒吧派对 | 饮料不离视线，不喝陌生人给的酒 |

### 常见骗局
| 骗局 | 手段 |
|------|------|
| 租房骗局 | 假房源、假房东、要求提前汇款 |
| 大使馆诈骗电话 | "你有一份重要文件"——正宗骗局 |
| 换汇骗局 | 私人换汇汇率好但可能是陷阱 |
| 网络购物 | 二手群买东西先付钱后拉黑 |

## 🏠 居家安全

- 出门锁好门窗（英国一楼窗户很多没防盗网）
- 买个小型保险箱放护照和贵重物品
- 烟雾报警器别拆，厨房做饭注意通风
- 租房最好有 Contents Insurance（财物保险，一年£50-100）

---

## 👮 遇到问题怎么办

| 情况 | 应对 |
|------|------|
| 被偷抢 | 立即报警 999，联系学校国际办公室 |
| 护照丢失 | 报警后联系使领馆补办旅行证 |
| BRP 丢失 | Gov.uk 在线报失，£19 补办 |
| 被歧视/hate crime | 报警 999 或 101，学校也会介入 |
| 心理困扰 | 联系学校 Counselling 或 NHS |

---

> 🛡️ **底线原则**：人身安全 > 财产安全。遇到危险先保命，东西丢了能再买。
